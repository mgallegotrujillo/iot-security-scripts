## Overview

Collection of practical security scripts for auditing IoT and OT networks. Includes Modbus TCP device discovery, ARP-based network mapping, port scanning, and automated Markdown report generation.

Built for security engineers who need fast, reliable visibility into industrial networks without enterprise tooling.

## Features

- Modbus TCP device scanner with function code probing
- ARP discovery for IoT device mapping
- Bash-based network audit with Markdown report output
- Nmap wrapper tuned for OT-safe scanning
- Modular structure — use each script independently
